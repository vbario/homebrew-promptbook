# PromptBook Homebrew tap

The terminal with a dockable, runnable notebook — for macOS 13 or later, signed and notarized.

```sh
brew install --cask vbario/promptbook/promptbook
```

Upgrading and removing:

```sh
brew upgrade --cask promptbook
brew uninstall --cask --zap promptbook
```

The cask installs the same disk image published at
[vbario/promptbook-releases](https://github.com/vbario/promptbook-releases/releases).
